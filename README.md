<p align="center">Easily Book Appointments with Your Favorites Barbers</p>

## Pre-requisites

Before you begin, you will need to have the following tools installed on your machine:

- `NPM` or `Yarn`

  - `NPM`:<br />
    recommend version 6.14

    Installing `Node` will also install `NPM` automatically.

  - `Yarn`:<br />
    recommend version 1.22<br />
    required version 1.\*

    `Yarn` can be installed from `https://classic.yarnpkg.com/en/`.

## Running the APP

- Install the project dependencies:

```sh
yarn install
# or
npm install
```

- Build the APP:

```sh
yarn android
# or
npm run android
```

- Start the APP:

```sh
yarn start
# or
npm start
```

## Testing the APP

- Install the project dependencies:

```sh
yarn install
# or
npm install
```

- Run tests:

```sh
yarn test
# or
npm run test
```

- Run coverage:

```sh
yarn test:coverage
# or
npm run test:coverage
```
